# SBA 316 - Goal Tracker

A single-page web application that lets users add goals, mark them complete, and clear the list.

## Features
- Add goals with HTML + JavaScript validation
- Mark goals as Done/Undo
- Clear all goals
- Dynamic DOM updates (create/append/update text/classes/attributes)
- Uses basic BOM methods

## Requirements Covered (SBA 316)
- getElementById + querySelector caching
- Parent/child navigation (parentNode, firstElementChild)
- Loop through collection (querySelectorAll + forEach)
- createElement + appendChild
- Template content using cloneNode
- Update textContent and CSS classes on user events
- Attribute changes based on user actions
- Multiple event listeners (submit, click)
- BOM usage (Date, alert, setTimeout)

## Tech
HTML, CSS, JavaScript
